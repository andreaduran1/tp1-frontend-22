En este trabajo práctico, nos hemos enfocado en desarrollar una aplicación web utilizando HTML, CSS y JavaScript. Hemos implementado diversas funcionalidades para mejorar la experiencia del usuario, como la navegación fluida entre páginas, el diseño responsivo para adaptarse a diferentes dispositivos y la integración de elementos interactivos. Además, hemos seguido buenas prácticas de desarrollo web para asegurar que nuestro código sea limpio, eficiente y fácil de mantener. A lo largo del proyecto, hemos aprendido a trabajar en equipo, a gestionar el tiempo de manera efectiva y a resolver problemas técnicos de manera colaborativa
